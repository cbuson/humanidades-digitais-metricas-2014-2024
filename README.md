# humanidades-digitais-metricas-2014-2024
Análisis modelométrico basado en datos de Lens.org (2014–2024). El corpus HDLENS2014-2024 reúne publicaciones en humanidades digitales, procesadas con Python e IRaMuTeQ para estudiar coocurrencias, métricas de impacto y evolución temática en la producción científica.

Humanidades Digitales Métricas 2014–2024

Descripción breve:
Análisis modelométrico basado en datos de Lens.org (2014–2024). El corpus HDLENS2014-2024 reúne publicaciones en humanidades digitales, procesadas con Python e IRaMuTeQ para estudiar coocurrencias, métricas de impacto y evolución temática en la producción científica.

Contenido del repositorio

HDLENS2014-2024.txt
Corpus principal depurado en texto plano, generado a partir de las exportaciones de Lens.org.
Incluye títulos, resúmenes y metadatos de publicaciones académicas sobre Humanidades Digitales.

estudio_metricas_muestras.xlsx
Planilla con indicadores derivados del corpus:
frecuencia de términos, citas acumuladas, coautorías, redes institucionales y evolución temática.

RHD-art-plantilla-modelometricasV06012025v2.pdf
Documento principal con la metodología completa, gráficos finales y discusión de resultados.


Herramientas utilizadas

Lens.org — Fuente de datos bibliográficos.

Python — Limpieza, lematización y análisis métrico.

IRaMuTeQ — Segmentación temática y redes de coocurrencia.

VOSviewer / Gephi — Visualización de redes y mapas conceptuales.
